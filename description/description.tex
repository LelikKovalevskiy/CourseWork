\section{Цели и задачи курсовой}

Основной задачей курсовой работы является создание приложения для создания анимации на базе формата SVG и как следствие ознакомление с обработкой xml-файлов и 2D графикой.

\section{Видение идеального приложения}

Идеальное приложение должно предоставлять возможности для для рисования геометрических фигур, линий, изменения их цвета и т.д.(похожая функциональность с paint).
Создание самой анимации должно осуществляться с помощью добавления сцен и установки времени перехода между ними(общая концепция похожа на Adobe Flash).

\section{Архитектура идеального приложения}
Функциональные части:
\begin{enumerate}

\item Управление графическими элементами на экране
\item Запись графики в SVG формат и сохранение
\item Чтение SVG формата и отображение его в виде 2D графики.

\end{enumerate}
Для каждой из этих функций потребуется:
\begin{enumerate}

\item Главное окно с набором кнопок и прочих элементов управления.
\item Область для отображения графики, классы графических элементов (эллипс, линия, точка и т.д).
\item Класс, взаимодействующий с графическими элементами, создающий DOM-дерево, записывающий и сохраняющий это дерево в файл.
\item Парсер, создающий DOM-дерево документа и соответствующие классы графических элементов.

\end{enumerate}
\section{Планы на реализацию в этом году}
Планируется создание  простого редактора для создания графических элементов (по типу Paint), реализация создания сцен, перехода от сцены к сцене, а также запись получившейся анимации в SVG-формат.
